# このWebアプリについて

### このWebアプリは歩きスマホを検知するためのアプリです
**実行環境**
- Window11
- Python 3.12
- Phyphox(スマホアプリ)

#### また, Phyphoxの接続の関係上ローカルで動かすWebアプリです.


## Webアプリについて

### このWebアプリできること
  - 歩きスマホかどうかの検知
  - 判定するためのモデルの作成
  - 歩きスマホ検知後にメールの送信(設定の必要あり)
 
### 実行方法
1.  このリポジトリをクローン(コピペで大丈夫)
  ```bash
  git clone -b master https://github.com/kiyo1204/AI_App
  ```
2.  メールアドレスの設定
  - メール送信には *.sreamlit/secrets.toml* の[email]の値の設定が必要です
  - 送信用のメールアドレス(Gmail)[^1]とアプリパスワードを設定してください
     [^1]: Gmailでないと動かないライブラリのため

3. 起動
   - まず, 必要なライブラリをインストールします
   ```bash
   pip install -r requirements.txt
   ```
   
   - 起動には以下のコマンドを実行します
   ```bash
   streamlit run main.py
   ```
   - エラーが出る場合はコマンドプロンプトのカレントディレクトリの確認

    
### 使い方
#### スマホでの手順
1. Phyphoxをダウンロードします
2. 「新しい実験の追加」で**線形加速**と**ジャイロスコープ**を選択します. センサーレートは60Hzにします(低すぎなければこれでなくても大丈夫).
3. PCと同じネットワークに繋いで「リモートアクセスを有効にする」を押します
     
#### PCでの手順
1. 最初にユーザー登録をします.
2. 次にモデル作成ページで歩きスマホ検知判定のためのモデルを作成します
3. モデルを作成出来たら保存します
4. リアルタイム処理ページにて先程保存したモデルとスマホのリモートアクセスで表示されたIPアドレスを入力します
